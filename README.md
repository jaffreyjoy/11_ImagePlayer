### Setup

 - Install npm packages
    ```bash
    > cd client
    > npm install
    ```
 - Install pip packages
    ```bash
    > cd server
    > pip install -r requirements.txt
    ```

### Run

 - Run front-end server
    ```bash
    > cd client
    > npm start
    ```
 - Run back-end server
    ```bash
    > cd server/src
    > uvicorn main:app --reload
    ```
